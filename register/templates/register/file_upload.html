How does validation work?

Metadata files will be sent to our servers for validation. Validation times can vary, but a metadata file will usually up to 20 seconds to a minute to finish validating (most of this is usually taken up by XSD validation).

The validation process involves:

  • Syntax validation
  • UTF-8 encoding validation
  • XSD validation
  • Type validation
  • Local ID validation
  • Namespace validation
  • Validating references to other metadata
  • Validating references to Space Physics Ontology terms

Additional checks will be performed additional checks depending on if metadata is being newly registered or is updated.
