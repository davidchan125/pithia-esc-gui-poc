Information on File Validation
How does validation work?

Metadata files must pass a validation process before they can be published in the e-Science Centre. The validation process is performed on our servers, and feedback is sent back to you as various checks are completed.

The validation process involves:

  • Syntax validation
  • UTF-8 encoding validation
  • XSD validation
  • Type validation
  • Local ID validation
  • Namespace validation
  • Validating references to other metadata
  • Validating references to Space Physics Ontology terms

Additional checks will be performed depending on if metadata is being updated or newly registered.

How long does validation take?

Validation times can vary, but a metadata file will usually take up to 20 seconds to a minute to finish validating (most of this is usually taken up by XSD validation).
