How long does validation take?

Validation times may vary, but a metadata file will usually take up to 20 seconds to a minute to finish validating (most of this is usually taken up by XSD validation).

  • Why do I have to validate my metadata files?

    Metadata files are validated to ensure that the e-Science Centre will be able to display the contents of each file and that links to ontology terms and other metadata registrations will work.

  • How does validation work?

    The validation process is performed on our servers, and feedback for each metadata file is sent back to you as various checks are completed.

    The validation process checks that each metadata file:

    • Uses XML 1.0 syntax and UTF-8 encoding
    • Shares the same typing for the current metadata file upload category (e.g. the metadata file has "Organisation" as the root element name, if you are registering it as/updating an organisation)
    • Uses a valid local ID and namespace (see the Data Resource Registration Guide (Chapter 4.1.1 Local ID and Chapter 4.1.2 Choice of namespace) for details)
    • Uses the local ID of the metadata file as the file name
    • Adheres to the rules and constraints defined in the relevant XSD schemas (see the PITHIA metadata models)

    If URL references to other metadata registrations and/or Space Physics Ontology terms are made in the metadata file, validation will verify that the registrations/terms at those URLs are currently registered in the e-Science Centre/Space Physics Ontology.

    Additional checks will be performed depending on if you are registering or updating.
